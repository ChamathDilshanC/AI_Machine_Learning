# MNIST දත්ත එකතුව මත Softmax Regression

මෙම ව්‍යාපෘතිය මගින් **MNIST දත්ත එකතුව (MNIST dataset)** බාවිතා කරමින් අත් අකුරින් ලියන ලද ඉලක්කම් (handwritten digits) වර්ගීකරණය කිරීම සඳහා **Softmax Regression** (බහු-පංති වර්ගීකරණය සඳහා ලොජිස්ටික් ප්‍රතිපායනය සාමාන්‍යකරණය කිරීමක්) භාවිතා කරන ආකාරය පෙන්වා දෙයි.

## ව්‍යාපෘතියේ අන්තර්ගතය (Project Overview)

මෙම සටහන් පොත (notebook) යන්ත්‍ර ඉගෙනීමේ (machine learning) ව්‍යාපෘතියක සම්පූර්ණ ක්‍රියාවලිය ආවරණය කරයි:

1.  **දත්ත ලබා ගැනීම (Data Loading)**: `sklearn.datasets.fetch_openml` භාවිතා කරමින් MNIST දත්ත එකතුව ලබා ගැනීම.
2.  **දත්ත ගවේෂණය (Data Exploration)**: ආදානය (input) තේරුම් ගැනීම සඳහා උදාහරණ පින්තූර දෘශ්‍යමාන කිරීම (Visualizing) සහ දත්තවල ප්‍රමාණ පරීක්ෂා කිරීම.
3.  **දත්ත සැකසීම (Preprocessing)**: ආකෘතියේ ක්‍රියාකාරිත්වය වැඩි දියුණු කිරීම සඳහා පික්සල අගයන් (pixel values) 255.0 න් බෙදා 0-1 පරාසයට සකස් කිරීම.
4.  **ආකෘතිය පුහුණු කිරීම (Model Training)**: `lbfgs` solver සමඟ `sklearn.linear_model.LogisticRegression` භාවිතා කරමින් Softmax Regression ආකෘතියක් පුහුණු කිරීම.
5.  **ඇගයීම (Evaluation)**: පරීක්ෂණ දත්ත කුලකය (test set) මත නිරවද්‍යතාවය (accuracy) මැනීම සහ ආකෘතියට වැරදුනු තැන් හඳුනා ගැනීමට Confusion Matrix එකක් භාවිතා කිරීම.
6.  **අනාවැකි පල කිරීම (Prediction)**: ආකෘතියේ ක්‍රියාකාරිත්වය පෙන්වීමට අහඹු ලෙස තෝරාගත් දත්ත සාම්පල සඳහා අනාවැකි පල කිරීම.

### අවශ්ය මෘදුකාංග සහ පුස්තකාල (Prerequisites)
